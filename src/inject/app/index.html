This Urdu Keyboard enables you to easily type Urdu. You can use your computer keyboard or mouse to type Urdu letters with this keyboard.

Pressing Esc on the Urdu Phonetic keyboard layout will toggle the mouse input between virtual QWERTY keyboard and virtual Urdu keyboard. The key will also turn on/off your keyboard input conversion. Pressing Esc on your keyboard has the same function.

Press Shift for additional Urdu letters that are not visible on the keyboard.
